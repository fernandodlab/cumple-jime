Colocá tus archivos dentro de /assets con estos nombres (o ajustá el HTML):
- video1.mp4, video2.mp4   (formato recomendado: H.264 + AAC)
- poster1.jpg, poster2.jpg (opcional: miniaturas para los videos)
- photo1.jpg ... photo8.jpg (galería; podés borrar o agregar más <img> en el HTML)

Cómo usar:
1) Abrí index.html en el navegador para ver la página.
2) Reemplazá los archivos de assets por tus videos y fotos.
3) Para compartir:
   - Mandá la carpeta a alguien (por WhatsApp no siempre abre bien). 
   - O subila a Netlify, Vercel o GitHub Pages (gratis) y compartí el link.
